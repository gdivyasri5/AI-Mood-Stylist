# 👗 AI Mood Stylist

**A Professional Personalized Fashion Assistant**

## 📌 About The Project
Finding the right outfit for your mood, body size, and specific events across multiple shopping apps can be exhausting. **AI Mood Stylist** solves this by providing an interactive, questionnaire-driven platform that generates instant style recommendations and direct shopping links based on your unique profile.

Developed by **Divya Sri**, a final-year B.Tech student at JNTUH UCES, this project showcases interactive frontend logic, state management, and user-centric application design.

## 🚀 Live Demo
[**Click here to try the AI Mood Stylist!**](https://gdivyasri5.github.io/AI-Mood-Stylist/)

## ✨ Features
* **Interactive Style Profile Wizard:** Dynamically captures user size, mood, and event type.
* **Smart Recommendations:** Maps user inputs to tailored outfit suggestions (dress, hair, and accessories).
* **Real Marketplace Deep Links:** Automatically generates targeted search links for Amazon and Myntra based on the chosen event.
* **Responsive UI:** Clean, modern interface built with Tailwind CSS for a seamless mobile and desktop experience.

## 🛠️ Tech Stack
* **Frontend:** HTML5, JavaScript (Vanilla)
* **Styling:** Tailwind CSS
* **Deployment:** GitHub Pages
